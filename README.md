# ☕ Coffee Website

A modern and responsive coffee shop website built using HTML, CSS, and JavaScript.  
This project showcases a stylish coffee landing page with sections like home, about, menu, tesimonials and contact.

## 🚀 Features
- Responsive coffee shop design
- Modern UI layout
- Hero section with call-to-action
- Coffee menu section
- About coffee section
- Testimonials section
- Contact form
- Smooth scrolling navigation

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
- Swiper.js (for slider)
